## GoSCA [![Go 1.17](https://img.shields.io/badge/Go-1.17-blue.svg)](https://go.dev/)

GoSCA scans a Go project for vulnerable dependencies.

Usage
```bash
./gosca --work-dir /path/to/workdir
```
